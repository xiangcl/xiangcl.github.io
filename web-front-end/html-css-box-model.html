css核心内容——流

标准流/非标准流

  • 流：在现实生活中就是流水，在网页设计中就是指元素（标签）的排列方式（默认情况下，我们的元素，向网页的左上角流动）；
  • 标准流：元素在网页中就像流水，排在前面的元素（标签）内容在前面出现，排在后面的元素在后面出现。
  • 非标准流：当某个标签脱离了标准流（比如因为相对定位）排列，我们统称为非标准流排列。
